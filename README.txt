[ DESCRIPCIÓN ]
Navaja Suiza PDF es una suite de ofimática profesional diseñada para manipular documentos PDF de forma rápida, segura y local. 
No requiere internet para funcionar y procesa todos los archivos en tu propio equipo para garantizar la privacidad.

[ HERRAMIENTAS INCLUIDAS ]
1.  Fusión y Conversión (Word a PDF, Imágenes a PDF, Unir PDFs).
2.  Optimización (Comprimir PDF, PDF a Word).
3.  Seguridad (Proteger con contraseña, Desbloquear, Marca de Agua).
4.  Utilidades Avanzadas (OCR/Escáner, Extraer Páginas, Rotar, Limpiar Metadatos).

[ REQUISITOS DEL SISTEMA ]
* Sistema Operativo: Windows 10 o Windows 11 (64 bits).
* Espacio en Disco: 200 MB libres.
* Requisito Adicional: Microsoft Visual C++ Redistributable (x64) 
  (Si el escáner no funciona, el instalador lo solicitará).

[ INSTRUCCIONES DE INSTALACIÓN ]
1. Ejecute el archivo "Instalador_NavajaSuiza_v3.exe".
2. Siga los pasos del asistente de instalación.
3. Al finalizar, encontrará el acceso directo en su Escritorio.

[ NOTAS LEGALES ]
Este software utiliza las librerías de código abierto PDFSharp y Tesseract OCR.
